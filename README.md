# NLteachers - 永不离开的支教老师

一个基于uni-app开发的跨平台教育类应用，旨在为支教老师和学生提供持续的在线教育支持。

## 功能特点

- 实时聊天功能
- 智能对话系统
- 成语接龙游戏
- 多平台支持（iOS、Android、H5）

## 技术栈

- uni-app
- Vue.js
- FirstUI组件库

## 开始使用

1. 克隆项目
```bash
git clone https://github.com/[your-username]/NLteachers.git
```

2. 安装依赖
```bash
npm install
```

3. 运行项目
```bash
npm run dev:h5          # 运行H5版本
npm run dev:app-plus    # 运行App版本
```
